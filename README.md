# Zarla-Browser
Zarla Browser is a powerful but very lite weight Browser solution that is Open source This is an alternation of Chromium, Firefox.


## What is some benefits of Zarla?
* Fast Speed
* Lite Weight
* Easy Source Code
* open Source


## When Will this come out?
Currently this is in development the estimated Release is currently being devised at the moment

# Can you describe what Zarla really is in full details

Zarla is a open source browser that is main Goal is to devised against Other Source code browsers such as Chromium, Firefox, Brave and many others
